# AoTGrapple

[UE4SS](https://github.com/UE4SS-RE/RE-UE4SS/releases) Lua mod for Palworld.

AoTGrapple plays the Attack on Titan opening theme from the first season when you use any grappling gun. 

You can even change the audio file that plays; simply replace `grapple.wav` with another WAV file using the same filename.

---

Requirements:

- Palworld
- [UE4SS](https://github.com/UE4SS-RE/RE-UE4SS/releases)
- Windows

Install:

1. Extract `AoTGrapple` into:
   `Palworld\Pal\Binaries\Win64\ue4ss\Mods`
2. Confirm this file exists:
   `Palworld\Pal\Binaries\Win64\ue4ss\Mods\AoTGrapple\Scripts\main.lua`
3. Start Palworld.
